使用Pelican与Github Pages搭建博客

四 12 三月 2015
By Gc

1.前言

Pelican是一个由Python编写的静态页面生成器。

官方网站:Pelican

特点

* 易部署
* 可用reStructuredText、Markdown等格式进行编写。
* 使用命令行界面生成页面

为啥叫“Pelican”？

Pelican是calepin(在法语中意为“笔记本”)的同字母异序词。

2.下载安装

git

Python

pip

Pelican

markdown

  1. 安装Pelican pip install pelican
  2. 安装Markdown pip install markdown

3.搭建

创建文件夹

mkdir blog
cd blog

执行pelican-quickstart命令，将会提示博客的配置项，就算手抖选错，也可在pelicanconf.py文件中进行修改。

命令执行后，出现pelican框架，如下所示： 

blog/
├── content                # 存放输入的markdown或RST源文件
│   └── (pages)            # 存放手工创建的静态页面，可选
├── output                 # 存放最终生成的静态博客
├── develop_server.sh      # 测试服务器
├── Makefile               # 管理博客的Makefile
├── pelicanconf.py         # 配置文件
└── publishconf.py         # 发布文件，可删除

4.写博文

博主使用的Mou创建.md，保存在content文件夹中。

源文件头部Title、Date、Category是必须写的

Title: 文章标题
Date: 创建日期
Modified: 修改日期
Category: 文章分类，标志本文处于该分类下
Tags: 文章标签，标志本文处于该标签下
Slug: URL中该文章的链接地址
Author: 作者

写完markdown文件后，回到blog目录下。 执行make html命令生成博客（make html命令把刚写的博文生成html，并放到output目录下）

执行make serve命令，在浏览器中输入http://localhost:8000即可看到博文效果。

5.主题

下载Pelican主题库

git clone https://github.com/getpelican/pelican-themes.git

打开pelicanconf.py配置文件，添加主题(本博客使用主题为gum)

THEME = 'pelican-themes/gum'

6.预览你的博客

新开一个命令行界面，运行下面的命令，将使用Python内建的服务器进行浏览：

cd ~/projects/yoursite/output
python -m SimpleHTTPServer

在浏览器输入http://localhost:8000进行预览

7.发布

行百里者半九十。

进入output目录下，执行以下命令。

git init
git add .
git remote add origin https://github.com/Gnahceg/Gnahceg.github.io
git pull origin master
git commit -m "First Blog"
git push origin master

最后，访问http://gnahceg.github.io即可。
