装配 mac 小记

触控板

最先学会使用的便是触控板，在系统偏好设置里的触控板设置里，有详细的教程。

HomeBrew

移步官网JustHomeBrew

  • 安装
    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

  • 使用
    brew install wget

  • 可使用 Homebrew 安装的：

    • git
    • python
    • macvim
  • 使用 HomeBrew Cask 安装：
    例：brew cask install google-chrome
    • google-chrome
    • iTerm2
    • mou
    • rescuetime
  • Tips
    输入brew info XXX可查看安装包的信息

码畜神器---Alfred

安装与简介，可移步借助 Alfred 2 的 Workflows 功能可以做哪些好玩的事情？。
Github上有一个AlfredWorkflow，安装后可在Alfred中通过wf命令安装Workflows。

打洞翻墙之术

Across the Great Firewall, you can reach every corner in the world.

  • 安装GoAgentX

  • ProxyChains-NG

    • 安装
      使用Homebrew安装 ProxyChains-NG ，实现命令行代理。
      brew install prochains-ng
    • 配置
      编辑配置文件 mvim /usr/local/etc/proxychains.conf 在 [ProxyList] 下面（也就是末尾）加入代理类型，代理地址和端口
    • 使用
      proxychains4 XXX

方便好用的插件

  • aText 输入增强工具，可用缩写代替经常输入的文字
  • CheatSheet 快捷键提示，长按command键即可提示组合输入
  • ClipMenu 剪切板管理工具
  • iStat Menus 显示网速，CPU等信息
  • Spectacle 方便调整窗口大小
  • XtraFinder Finder扩展增强
  • RescueTime 纪录电脑使用情况
  • Caffeine 保持屏幕常亮

Tips

  1. 使用触发角来锁屏

  2. Macintosh自带发音功能，进入Terminal，输入say XXXXXX为想说的内容，也可在系统偏好设置里的听写与语音设置发音选项。

  3. 快捷键：command + ,键可查看软件的偏好设置。
