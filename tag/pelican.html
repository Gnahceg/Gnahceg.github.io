  1. 使用Pelican与Github Pages搭建博客

    四 12 三月 2015
    By Gc

    1.前言

    Pelican是一个由Python编写的静态页面生成器。

    官方网站:Pelican

    特点
    * 易部署
* 可用reStructuredText、Markdown等格式进行编写。
* 使用命令行界面生成页面
    为啥叫“Pelican”？

    Pelican是calepin(在法语中意为“笔记本”)的同字母异序词。

    2.下载安装

    git

    Python

    pip

    Pelican

    markdown

    1. 安装Pelican pip install pelican
    2. 安装Markdown pip install markdown

    3.搭建

    创建文件夹

    mkdir blog
cd blog

    执行pelican-quickstart命令，将会提示博客的配置项，就算手抖选错，也可在pelicanconf.py文件中进行修改。
